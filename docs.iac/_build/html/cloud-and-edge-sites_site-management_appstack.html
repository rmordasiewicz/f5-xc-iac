Datacenter Edge

Create Appstack Site

  • In Cloud and Edge Sites, navigate to Manage ‣ Site Management ‣ App Stack Sites and click Add App Stack Site

  • Fill in the form using the following values, and click

    1. Name: Provide a unique <site-name>

    2. Basic Configuration ‣ Generic Server Certified Hardware: select kvm-voltstack-combo.

    3. Basic Configuration ‣ Order Master Node: enter a unique <node-name>.

    4. Basic Configuration ‣ Geographic Address: enter a <Geographic Address>.

    5. Basic Configuration ‣ Coordinates ‣ Latitude: enter site coordinates <47.605199>.

    6. Basic Configuration ‣ Coordinates ‣ Longitude: enter site coordinates <-122.330996>.

  • The site appears in the list and indicates Waiting for Registration

Replace highlighted values in appstack_site.json
