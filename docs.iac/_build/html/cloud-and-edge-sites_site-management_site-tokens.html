Site Tokens

  1. Navigate to Manage ‣ Site Management ‣ Site Tokens and click Add K8s Cluster

  2. Name the token, and click Add Site Token

  3. The site token appears with the UID field, copy this value for registering the CE node.

Replace highlighted values in token.json

{
  "metadata": {
    "name": "<site-name>-token",
    "namespace": "system"
  },
  "spec": {}
}

$ vesctl configuration apply token -i token.json
Created
