Deploy CE

  • Using a browser log-in to the CE https://<CE-IP-ADDRESS>:65500 with the default username:admin and password:Volterra123, and click Configure Now

  • Fill in the form using the following values, and click :bdg-primary`Save configuration`

    1. Device Configuration ‣ * Cluster name: Provide the site cluster name

    2. Device Configuration ‣ Hostname: provide a unique hostname.

    3. Device Configuration ‣ Certified Hardware: select kvm-voltstack-combo

    4. Location ‣ Latitude: provide <47.605199>

    5. Location ‣ Longitude: provide <-122.330996>

  • The CE advances to the Approval stage.

  • SSH or access the console with default username: admin and default password Volterra123

    4. Enter latitude and longitude

  • Edit the ce-register.json and change values

    {
  "token": "<token>",
  "cluster_name": "<site-name>",
  "hostname": "<node-name>",
  "latitude": "<latitude>",
  "longitude": "<longitude>",
  "certified_hardware": "kvm-volstack-combo",
  "primary_outside_nic": "eth0"
}

  • Run the following curl command to remotely configure the CE node.

    $ curl -k \
  -u "admin:Volterra123" \
  -H 'Content-Type: application/json' \
  -d @ce-register.json \
  https://10.1.1.5:65500/api/ves.io.vpm/introspect/write/ves.io.vpm.config/update
